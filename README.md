
## Dependencies

- `sounddevice`: Low-latency audio playback
- `numpy`: Audio signal generation
- `soundfile`: Audio file handling
- `pydub`: Audio processing

## System Requirements

- Python 3.8+
- Audio output device
- Operating System: Windows/macOS/Linux
